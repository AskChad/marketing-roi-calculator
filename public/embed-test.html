ROI Calculator Embed Test

This page demonstrates how the ROI Calculator can be embedded in external websites.

How to embed this calculator on your website:

  1. Log into the Demo Calculator as an admin
  2. Click the "Share / Embed" button in the top right
  3. Copy the embed code provided
  4. Paste it into your website's HTML
  5. Adjust width and height as needed

Note: The embedded calculator works without requiring authentication, making it perfect for prospects to use.
